Навики фронт-енд розробника

Фронт-енд розробник (з англ. «Front-end developer») - це програміст, що займається розробкою користувальницького інтерфейсу, тобто зовнішній публічній частині сайту в браузері. Головне завдання фронт-енд розробника - зробити максимально зручним взаємодію користувачів з сайтом або веб-додатком. Основні інструменти фронт-енд розробника в web: JavaScript, HTML, CSS.
HTML5 глобальні атрибути (id, tabindex, hidden, data-), нові компоненти введення(date/time, email, url, search, number, range, tel, color), нові теги (article, audio, сanvas, header, footer та ін.), валідна, кроссбраузерна верстка
Властивості CSS3 : animation, transition, transform та ін. Flexbox, Grid system, Media queries та ін.
JavaScript ООП, DOM, AJAX, бібліотеки JQuery
